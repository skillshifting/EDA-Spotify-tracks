# Анализ данных Spotify: Исследование популярности треков (EDA)

Проект содержит пример первичного анализа данных (EDA) и исследование динамики популярности музыкальных треков в чартах Spotify для выявления ключевых закономерностей.

### Что сделано:

* **Предобработка данных:** Загрузка и первичная очистка датасета, работа с типами данных и подготовка временных рядов для анализа.
* **Feature Engineering:** Обработка дат для построения временных графиков, фильтрация данных по популярным трекам.
* **Исследовательский анализ (EDA):** Визуализация динамики изменения позиций (`daily_rank`) треков в чартах с течением времени. Построены линейные графики для сравнения популярности с использованием `matplotlib`.
* **Статистический подход:** Использование библиотек `scipy` и `pandas` для агрегации данных и выявления ключевых характеристик треков.

### Результаты анализа:

* Выявлена динамика изменения позиций топовых треков в чартах.
* Настроена система визуализации, позволяющая сравнивать треки между собой по их «живучести» в рейтингах.
* Сформирован пайплайн для быстрой фильтрации и визуализации любого трека из базы по его названию.

### Стек технологий:

* Python, Pandas (манипуляции с данными), NumPy.
* Matplotlib, Seaborn (визуализация графиков).
* Scipy (статистические вычисления).
* Jupyter Notebook (среда разработки).
